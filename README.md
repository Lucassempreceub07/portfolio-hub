# Portfólio Hub — Lucas Cerqueira

Portfólio pessoal e profissional hospedado no GitHub Pages, desenvolvido como projeto acadêmico para a disciplina de Arquitetura de Soluções — UniCEUB 2026.

## Tecnologias Utilizadas

- **GitHub Pages** — Hospedagem estática gratuita
- **GitHub Actions** — Pipeline de CI/CD automatizado
- **Google Forms** — Integração com Google Workspace para formulário de contato
- **Google Gemini 1.5 Flash** — IA utilizada para arquitetura e planejamento do projeto

## Estrutura do Projeto

```
portfolio-hub/
├── .github/
│   └── workflows/
│       └── deploy.yml   # Pipeline de deploy automático
├── index.html           # Página principal do portfólio
└── README.md            # Este arquivo
```

## Como funciona o deploy

A cada `push` na branch `main`, o GitHub Actions automaticamente publica a versão atualizada do site no GitHub Pages. Não é necessário nenhum comando manual após o envio do código.

## Atribuição de IA

Este projeto foi desenvolvido com assistência do **Google Gemini 1.5 Flash**, utilizado para:
- Arquitetura da solução de integração GitHub + Google Workspace
- Planejamento das fases de implantação
- Geração do modelo RBAC de segurança
- Documentação técnica do plano de implantação

Todas as respostas geradas pelo Gemini foram revisadas criticamente antes da implementação, conforme exigido pelas diretrizes acadêmicas do professor Marcelo Carboni Gomes.
